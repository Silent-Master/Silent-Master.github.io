开关电源的损耗计算

开关电源损耗计算(以Buck为例，包含同步和不同步)

本篇文章将以一个具体的拓扑为例，展开对开关电源损耗分析，并给出较为Concise的计算方式。之前一直有一个很好的想法，但由于时间原因一直没有实施，为此之后将从电力电子的最基础的Topology开始，也就是BUCK，然后从之延伸出Power Electronics的方方面面。

以下图的同步整流为例(图片来自于TI)，Synchronous Buck的Fundamental Element包括输入电容，一对开关管(HS MOSFET and LS MOSFET,这里忽略掉图片中的Rsense，通常不需要),电感，输出电容。从工作模态来说，当HS管开通，LS管关断，Vin和输入电容能量经过电感传递到副边，而当HS管关断，LS管开通，电感中的能量由于不能突变(电感电流不能突变)，能量由输出电容和电感传递给Vo，Syn Buck利用低位的MOS管代替二极管以降低低压大电流条件下的功率损耗。以VR应用场景下的Mosfet导通电阻4mΩ为例，在40A应用环境下，若用二极管作为下管(0.7V导通电压为例)，损耗为28W，而Mosfet的损耗仅为6.4W。尽管降低功耗是使用同步整流管的强烈动机，但真正使用同步整流管时也需要注意，当其作为整流管时，会阻碍电流从漏极流向源极，同时在Syn Buck等电路中，两个开关管的关断需要保证一定的死区时间。在此器件，电感电流仍需要一个流通路径，电流通过整流管的体二极管进行续流。因此，当HS管关断时，电感电流先转移到同步整流管的体二极管，然后在死区时间结束时，同步整流管导通并将体二极管旁路。这会带来一个新的问题，MOSFET的体二极管正向压降通常比肖特基二极管的正向压降要高，因此死区时间需要尽可能短，否则损耗加大。

从系统层面来看，当Syn Buck处于轻载时，不会出现传统Buck的DCM形式，整流管的使用会让电流反方向流动以使得Syn Buck工作在CCM模式。CCM一个显而易见的好处是能够让系统保持线性，并且不会对环路补偿产生影响，缺点是会让电感电流为负，从而使得其AC电流损耗相较于DCM稍高一些，因此市面上有的VR控制器会让系统在轻载时处于DCM模式，以使其在全范围获得较高的效率。

另外，同步整流管的使用也需要考虑到Perbias启动，当功率管在导通时，能量的流向可以是双向的，因此在1-D(D为占空比，为HS管开通时间)的时间内，若变换器试图启动，且输出端存在其它电压，电流会从输出端流向地，为此，为了实现快速prebias启激，各个控制芯片厂家也设置了不同的启动方式以避免此问题，同时降低开机时间。

上述浅谈了一下Synchronous Buck和Synchronous系列电路的工作原理及少量细节，这里需要回归一下主题，回到功率拓扑的损耗本身，为此，本文将以Syn Buck为例子，功率拓扑损耗分为以下几点：

  1. 开关管损耗

    1. 硬开关损耗

    2. 软开关损耗

    3. 同步整流管损耗

  2. PCB系列损耗

  3. 磁损耗

  4. 输入输出电容损耗

  5. 其它损耗

开关管损耗

以下图为例，开关管在功率拓扑中频繁切换只会工作在如下两种状态，一是硬开关状态，另一个是软开关状态。

硬开关表示开关管在开通和关断时(Transition Time)电压和电流有较大的交越面积，而软开关表示开关管在开通和关断时基本没有或没有明显的电压电流交越面积。根据不同的特性和技术发展历程，软开关可以分为以下几类：

  1. 零电压开关（Zero Voltage Switching，ZVS）

    在开通和关断时，电压先下降到零后，电流再缓慢上升到通态值。

  2. 零电流开关（Zero Current Switching，ZCS）

    在开通和关断时，电流先下降到零后，电压再缓慢上升到通态值。

  3. 准谐振电路

    利用谐振电路的特性(相位超前和滞后，主要源自于电路固有特性)，在开通和关断时实现零电压或零电流。

  4. 零开关PWM电路

    使用脉冲宽度调制即PWM技术(主要来自于调制分配)，实现开通和关断时的零电压或零电流。

硬开关损耗

以Syn Buck和传统Buck为例，HS管只有单相通路，关断时DS上始终存在压差，开通时又因为电感电流不能突变导致其基本在全时域下始终工作在硬开关状态。下图展示了HS管在开通关断时的Waveform：

  1. Conduction loss

    • 首先确定开关管流过的电流(列出时域表达式)

    • 计算导通阶段中的RMS电流

    • 考虑开关管温度对导通电阻影响以及开关管个数，计算处最终的导通损耗

    导通电阻的相关特性如下：

    1. 温度特性：不论是N沟道MOSFET（NMOS）还是P沟道MOSFET（PMOS），导通电阻RDS(on)都随着温度的升高而增大。这是因为MOSFET的导通电阻与其中的载流子迁移率有关，载流子迁移率越慢，导通电阻越大。MOSFET反型层中的电子和空穴迁移率随着温度升高而下降，因为温度升高会增加载流子在沟道中的散射几率。

    2. 阈值电压特性：阈值电压（VGS(th)）的绝对值随温度的升高而降低。这意味着在高温下，MOSFET更容易进入导通状态。

  2. Drive loss

    以TI驱动设计文档中图片为例：

    • Drive Loss主要是由开关管的栅极电荷造成

    • 考虑开关频率计算出最终的栅极电荷损耗，即驱动损耗

  3. Switching loss

    即开通关断交越面积损耗+漏极电荷所造成的损耗

    • 首先确定栅极驱动器(如有)的Rsource和Rsink电阻(开通为Source，关断为Sink，有的地方也分为HI和LO)以及开关管封装内的电阻

    • 将开通和关断的平台电压前后的充电放电阶段近似等效为线性充放电，以确定最终的充放电时间

    • 计算出开通关断过程中电流的表达式

    • 求积分计算最终开通和关断的损耗

    • 绘制出不同Vds电压下输出电容曲线，可以对输出电容取平均值再计算损耗(也可以取中间值，只是近似计算)

    • 汇总最终损耗

软开关损耗

软开关通过增加辅助电感和电容，达到开关管交越面积的减小(甚至为0)，从而基本消除了开关损耗和漏极电荷所造成的损耗，软开关情况下基本只用考虑驱动损耗。

同步整流管损耗

  1. Conduction loss
     同硬开关损耗

  2. Driver loss

    同硬开关损耗

  3. Deadtime Conduction loss

    • 考虑二极管温度的温度变化和死区时间计算最终的损耗

  4. Body-diode reverse loss

    • 利用电荷能量公式计算最终的损耗

PCB损耗

  • PCB Resistor loss

    • 利用电阻和电阻率公式计算(可折算成OZ，利用铜厚来计算)
    • 简化后的公式只包括路径长度、OZ常量、铜厚常量、线宽以及层数

  • Lead Resistor loss

    需要考虑铜的阻抗随温度变化情况

  • Vias Resistor loss

    • 利用电阻和电阻率公式计算(可折算成OZ，利用铜厚来计算)

    • 简化后的公式只包括路径长度、OZ常量、铜厚常量、线宽以及过孔个数

  • Current Resistor loss
    不同类型的电阻损耗不同，需要使用不同方法计算
